# Student Grade Calculator

## Description
Student Grade Calculator is a Java console application developed as part of DecodeLabs Project 2.

The program accepts marks for multiple subjects, validates the input, calculates the total marks, average, percentage, highest marks, lowest marks, assigns a grade, and displays the final result.

## Features

- Enter student name
- Enter number of subjects
- Input validation (marks between 0 and 100)
- Calculate total marks
- Calculate average marks
- Calculate percentage
- Display highest and lowest marks
- Grade calculation
- Pass/Fail status

## Grade Criteria

| Percentage | Grade |
|------------|-------|
| 90 - 100 | A |
| 80 - 89 | B |
| 70 - 79 | C |
| 60 - 69 | D |
| Below 60 | F |

## Technologies Used

- Java
- Scanner Class
- Loops
- Conditional Statements
- Methods

## How to Run

Compile:

javac StudentGradeCalculator.java

Run:

java StudentGradeCalculator
